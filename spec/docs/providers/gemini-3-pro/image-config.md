# Aspect ratio и resolution (imageConfig)

Источник:
- https://ai.google.dev/gemini-api/docs/image-generation#gemini-image-editing

## Параметры
- `imageConfig.aspectRatio` — строка из списка: `1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `4:5`, `5:4`, `9:16`, `16:9`, `21:9`.
- `imageConfig.imageSize` — `1K`, `2K`, `4K`.
